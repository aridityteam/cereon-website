A simple, and lightweight .NET IDE

Cereon is an Windows-only IDE that aims to recreate most basic IDE features that everybody mostly uses on Visual Studio. Powered by the .NET Framework 4.8 runtime, it delivers a lightweight, extensible, and high-performance development experience.

Built with Microsoft Visual Studio's Tools

Cereon is built on the same core technologies used in Microsoft Visual Studio, ensuring full compatibility with existing .NET projects.

Why even bother?

Because of the fact that Microsoft wants to integrate more AI features into their applications including regular Notepad, and obviously even Visual Studio + the fact that the new Visual Studio may likely use more performance on default settings. Cereon was made to try and make an lightweight IDE that doesn't ridiculously take up more performance usage on some important cases.
